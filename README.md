## 🛠️ Tech Stack & Design Architecture

This project is built using pure, high-performance web standards optimized for rapid loading and crisp rendering across mobile and desktop platforms:

*   **Engine:** Semantic HTML5 & Modern CSS3 Grid/Flexbox layouts.
*   **Aesthetic:** Y2K Brutalist / Streetwear design system featuring hard-drop shadows, high-contrast neon accents, and responsive fluid typography.
*   **Asynchronous Features:** Custom dynamic video-switching engine, modular toast notification systems, and client-side anti-cache meta integration.
